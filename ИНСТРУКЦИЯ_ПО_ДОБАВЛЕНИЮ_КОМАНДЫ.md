# Инструкция по добавлению команды "Журнал регистрации по объекту"

## Описание
Данная инструкция описывает, как добавить команду для открытия журнала регистрации с отбором по текущему объекту во все формы объектов, которые могут быть отобраны из ТЖ (транзакционного журнала).

## Созданные компоненты

### 1. Общий модуль `ЛогИ_ОбщийМодуль`
Расширен функциями:
- `ОткрытьЖурналРегистрацииПоОбъекту(ТекущийОбъект)` - открывает журнал регистрации с отбором
- `КомандаОткрытьЖурналРегистрации(Форма)` - команда для вызова из форм

### 2. Общий модуль `ЛогИ_ОбщийМодульВызовСервера`
Новый модуль для серверных вызовов:
- `КомандаОткрытьЖурналРегистрации(Форма)` - серверный вызов
- `ПолучитьДанныеДляОтбора(ТекущийОбъект)` - получение параметров отбора

### 3. Общая команда `ЛогИ_ОткрытьЖурналРегистрации`
- Название: "Журнал регистрации по объекту"
- Горячая клавиша: Ctrl+Shift+L
- Иконка: StdPicture.EventLogMonitor

## Добавление команды в формы объектов

### Для форм документов

1. Откройте форму документа в конфигураторе
2. Перейдите на вкладку "Команды"
3. Добавьте новую команду:
   - Имя: `ЖурналРегистрацииПоОбъекту`
   - Заголовок: `Журнал регистрации по объекту`
   - Действие: `ЛогИ_ОткрытьЖурналРегистрации`

4. Перейдите на вкладку "Элементы формы"
5. Найдите главное меню формы или панель навигации
6. Добавьте кнопку команды:
   - Имя: `КнопкаЖурналРегистрации`
   - Вид: `КнопкаКоманднойПанели`
   - Команда: `ЖурналРегистрацииПоОбъекту`

### Для форм справочников

Аналогично формам документов, добавьте команду в:
- Формы элементов справочников
- Формы списков справочников (для отбора по выделенному элементу)

### Для форм констант

1. В форме константы добавьте команду аналогично
2. Учтите, что для констант объект получается из свойства `Константа` формы

### Альтернативный способ - использование общей команды

Если общая команда `ЛогИ_ОткрытьЖурналРегистрации` правильно настроена в конфигурации, она должна автоматически появляться во всех формах объектов.

## Код для добавления в модуль формы (если требуется дополнительная логика)

```bsl
&НаКлиенте
Процедура ЖурналРегистрацииПоОбъекту(Команда)
    ЛогИ_ОбщийМодульВызовСервера.КомандаОткрытьЖурналРегистрации(ЭтотОбъект);
КонецПроцедуры
```

## Проверка работы

1. Откройте любую форму документа/справочника/константы
2. Найдите команду "Журнал регистрации по объекту" в главном меню или панели навигации
3. Нажмите на команду или используйте горячую клавишу Ctrl+Shift+L
4. Должен открыться журнал регистрации с отбором по событию "Лог. Изменение данных" и текущему объекту

## Применимые типы форм

- Формы документов (элемент)
- Формы справочников (элемент)
- Формы планов видов расчета (элемент)
- Формы констант
- Формы записей регистров сведений
- Формы записей регистров накопления

## Особенности

- Команда работает только для объектов, которые логируются системой
- Для новых (несохраненных) объектов отбор может работать некорректно
- Команда использует событие "Лог. Изменение данных" для отбора в журнале регистрации
